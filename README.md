# Project Exam 2

Exam Assignment with focus on react and bootstrap

## Description

This is a website where users can register, log in, create and book venues.

## Built With

- HTML
- CSS
- Javascript
- React
- Bootstrap

## Getting Started

### Installing

Make sure to have Node.js and npm installed.

Clone the repo

Install the dependencies:

npm install


### Running

npm run start
